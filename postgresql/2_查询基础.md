# 2.1 Select语句

**`查询`或`匹配查询`：通过Select语句查询并选取出必要数据的过程。**
* 子句之间插入空行会导致执行错误

--从表中输出某列：
```sql
	SELECT product_id, product_name, purchase_price
	  FROM Product;
```
* 查询结果中列的顺序和Select子句中的顺序相同。

--查询全部的列：
```sql
	SELECT *
	  FROM Product;
```
* 使用`*`，无法设定列的显示顺序。
